## Sobre o Projeto

A Vincere é uma marca de marketing digital focada em resultados reais para pequenas e médias empresas. Esta landing page foi criada para apresentar serviços, estratégia e trajetória profissional de forma impactante e objetiva.

**Serviços apresentados:**
- Edição de Vídeo — US$100/vídeo
- Social Media — US$250/mês
- Google Maps — US$25/mês

---

## Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript vanilla (scroll reveal, navbar, menu mobile)
- Google Fonts — Bebas Neue + Barlow
- Zero dependências externas
